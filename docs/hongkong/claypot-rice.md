![糯米飯](../images/claypot-rice.jpg){ width=600 }

## 材料
- 糯米
- 珍珠米 (比例 糯米:珍珠米，2:1)
- 水（比例 米：水+調味料，1:1 ）
- 生抽 1湯匙
- 蠔油 1湯匙
- 老抽 1茶匙
- 臘腸
- 蝦米
- 瑤柱
- 冬菇
- 乾蔥

## 做法
1. 臘腸，蝦米，瑤柱，冬菇，乾蔥 落鑊炒。  
2. 洗米，瀝乾。 洗多幾次，啲糯米好多粉，唔洗乾淨就會好粘。
3. 落調味料落水。  
4. 落晒啲料落去撈勻啲米。  
5. 加調味水。  
6. 開電飯煲，完成。
