---

![糯米飯](../images/claypot-rice.jpg){ width=600 }

## 材料
- 糯米
- 珍珠米
- 水（比例 1:1，米：水）
- 生抽 1湯匙
- 蠔油 1湯匙
- 老抽 1茶匙
- 臘腸
- 蝦米
- 瑤柱
- 冬菇
- 乾蔥
- 炒花生
- 蛋絲

## 做法
1. 所有料落鑊炒。  
2. 洗米，瀝乾。  
3. 落調味料落水。  
4. 落晒啲料落去撈勻啲米。  
5. 加調味水。  
6. 開電飯煲，完成。
