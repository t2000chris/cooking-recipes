# Reverse-sear Steak

## 材料
- 牛扒 1 塊
- 鹽
- 黑胡椒
- 牛油
- thyme

## 做法
1. 牛扒用鹽同胡椒醃半個鐘。
2. 焗爐開120度，焗到塊扒中心大約50度。
3. 攞出嚟大火煎。
4. 煎完後細火加牛油，用thyme上油。
