# 韓式泡菜鍋

## 材料
- 泡菜 400g
- 茄汁 30g
- 大白菜 半個
- 韓國辣椒粉 2湯匙
- 豬肉 600g
- 昆布 半塊
- 洋蔥 半個
- 蔥白 適量
- 豆腐 300g
- 味淋、白醋、魚露 自行調味
- 大蒜 20g
- 黑胡椒 自行調味
- 麻油 自行調味
- 蔥綠 20g （可選）
- 辣椒 10g （可選）

## 做法
1. 泡菜汁用嚟醃豬肉。
2. 豬油起鑊，炒泡菜。
3. 加入茄汁。
4. 加入大白菜、辣椒粉，唔夠油可以再加少少繼續炒。
5. 加入豬肉，炒5－8分鐘。
6. 加入1/3水，煮至豬肉熟。
7. 再加埋剩餘嘅水，放昆布、洋蔥、豆腐。
8. 加入味淋、白醋、魚露調味。
9. 最後加入大蒜、麻油、辣椒。
