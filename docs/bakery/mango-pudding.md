# 芒果布甸

![芒果布甸](../images/mango-pudding.jpg){ width=600 }

## 材料
- 芒果蓉 225ml（Sainsbury’s：KTC Kesar Mango Pulp）
- 鮮奶 225ml
- 鮮忌廉（whipping cream）50ml

### 啫喱液
- 魚膠粉 9g
- 砂糖 45g
- 水 170ml

### 裝飾
- 新鮮芒果粒 適量

## 做法
1. 將芒果蓉、鮮奶與鮮忌廉拌勻備用。
2. 將魚膠粉與砂糖混合後加入水，小火加熱至完全溶解（勿滾）。
3. 把啫喱液倒回芒果奶漿中拌勻。
4. 於杯底放入芒果粒，倒入布甸溶液。
5. 放雪櫃冷藏至凝固即可（建議至少 4 小時，隔夜更佳）。
