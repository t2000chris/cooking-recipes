# 狗狗蛋糕

![狗狗蛋糕](../images/dog-cake.jpg){ width=600 }

## 材料

### 8 吋圓形蛋糕
- 麵粉 96g  
- 梳打粉 2.4g  
- 植物油 20g  
- 花生醬 52g  
- 南瓜蓉 88g  
- 雞蛋 1 隻  

## 做法

1. 將所有材料混合均勻。  
2. 倒入蛋糕模中。  
3. 焗爐預熱至 170℃，焗至熟透（約 25–30 分鐘，可用竹籤測試）。  
4. 放涼後即可食用。
