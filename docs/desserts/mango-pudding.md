# 芒果布甸

**標籤**：dessert, pudding, hongkong, cold
**份量**：約 3–4 杯

## 材料

- 芒果茸 225 ml
- 鮮奶 225 ml
- Whipping Cream 50 ml
- 魚膠粉 9 g、砂糖 45 g、水 170 ml
- 芒果（切粒）適量

## 做法

1. 芒果茸、鮮奶與 Whipping Cream 拌勻。
2. 魚膠粉與砂糖拌勻，加水煮滾至溶。
3. 將兩者混合拌勻。
4. 杯內先放芒果粒，倒入布甸液，冷藏至凝固。
