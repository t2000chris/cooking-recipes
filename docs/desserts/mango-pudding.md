# 芒果布甸

## 材料
- 芒果茸 225ml （例：Sainsbury's - KTC Kesar Mango Pulp）
- 鮮奶 225ml
- Whipping Cream 50ml
- 魚膠粉 9g
- 砂糖 45g
- 水 170ml
- 芒果粒 適量

## 做法
1. 芒果茸、鮮奶、Whipping Cream 撈勻。
2. 魚膠粉同砂糖撈勻，加入水煮滾。
3. 將兩樣嘢撈勻。
4. 芒果粒放杯入面，倒入布甸溶液。
5. 放雪櫃冷藏至凝固。
